# 🏎️ F1 Reaction Challenge |Python + Tkinter

A Formula 1 inspired reaction time game built with **Python** and **Tkinter**.

The game recreates the iconic F1 starting lights sequence. Wait for the lights to go out, react as quickly as possible, and try to beat your personal best.

---

## ✨ Features

- 🏎️ Realistic five-light F1 start sequence
- 🚦 Randomized start timing
- ❌ False start detection
- ⏱️ Reaction time measured to milliseconds
- 🏆 Best time tracking
- 🔁 Play Again option
- 🏠 Main Menu
- 🎨 Simple Formula 1 inspired interface

---

## 🛠️ Built With

- Python 3
- Tkinter

---

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/nimsfernando/F1-Reaction-Game.git
```

2. Open the project folder

3. Run

```bash
python reaction_time.py
```
